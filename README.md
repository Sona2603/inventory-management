# Inventory Management System

A full-stack inventory management dashboard built using React, Django REST Framework, and Tailwind CSS.

## Features

- JWT Authentication
- Add / Edit / Delete Products
- Product Image Upload
- Dashboard Analytics
- Inventory Charts
- Search & Sorting
- Pagination
- Dark Mode
- CSV Export
- Sell Product Feature
- Low Stock Alerts
- Recent Activity Tracking

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios
- Recharts

Backend:
- Django
- Django REST Framework
- JWT Authentication
- SQLite

## Installation

```bash
python manage.py runserver
```

### Frontend

```bash

npm start
```
